# SPM
The Swap Package Manager for Swap Packages

## Dependencies 

* Colorama python module to print colors into terminal
* Tar


## Installing

To install, just run `install.sh` as root:
```bash
chmod +x install.sh
sudo ./install.sh
```

And there you go!

## Removing

To remove, just run `uninstall.sh` as root:
```bash
chmod +x uninstall.sh
sudo ./uninstall.sh
```
